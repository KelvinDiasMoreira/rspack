# Ecossistema

## Ferramentas Rstack

Rstack é uma toolchain JavaScript unificada e de alta performance baseada no Rspack, desenvolvida e mantida pela equipe Rspack.

### Rsbuild

<Tag color="geekblue">Ferramenta de build</Tag>

[Rsbuild](https://github.com/web-infra-dev/rsbuild) é uma ferramenta de build de alta performance baseada no Rspack. Oferece configurações de build pré-definidas e uma experiência de desenvolvimento pronta para uso.

### Rspress

<Tag color="geekblue">Gerador de site estático</Tag>
<Tag color="purple">React</Tag>

[Rspress](https://github.com/web-infra-dev/rspress) é um gerador de sites estáticos baseado em Rsbuild, React e MDX. Inclui um tema de documentação pronto para uso.

### Rslib

<Tag color="geekblue">Ferramenta para bibliotecas</Tag>

[Rslib](https://github.com/web-infra-dev/rslib) é uma ferramenta para desenvolvimento de bibliotecas JavaScript baseada no Rsbuild.

### Rsdoctor

<Tag color="geekblue">Analisador de builds</Tag>

[Rsdoctor](https://github.com/web-infra-dev/rsdoctor) é um analisador que mostra visualmente o processo de build.

### Rstest

<Tag color="geekblue">Framework de testes</Tag>

[Rstest](https://github.com/web-infra-dev/rstest) é um framework de testes desenvolvido para o ecossistema Rspack.

## Integrações da comunidade

### Angular Rspack

<Tag color="geekblue">Ferramenta de build</Tag>
<Tag color="purple">Angular</Tag>

[Angular Rspack](https://github.com/nrwl/angular-rspack) permite construir aplicações Angular com Rspack.

### Docusaurus

<Tag color="geekblue">Gerador de site estático</Tag>
<Tag color="purple">React</Tag>

[Docusaurus](https://docusaurus.io/) suporta Rspack como bundler desde a versão 3.6.

### Modern.js

<Tag color="geekblue">Framework web</Tag>
<Tag color="purple">React</Tag>

[Modern.js](https://modernjs.dev/en/) é um framework React progressivo baseado no Rsbuild.

### Next.js

<Tag color="geekblue">Framework web</Tag>
<Tag color="purple">React</Tag>

[Next.js](https://nextjs.org/) pode usar Rspack como bundler através do plugin `next-rspack`.

### Nuxt

<Tag color="geekblue">Framework web</Tag>
<Tag color="purple">Vue</Tag>

[Nuxt](https://nuxt.com/) introduziu suporte ao Rspack na versão 3.14.

### Nx

<Tag color="geekblue">Sistema de build</Tag>

[Nx](https://nx.dev/) oferece um plugin para gerenciar projetos Rspack.

### Rspeedy

<Tag color="geekblue">Ferramenta de build</Tag>
<Tag color="purple">Lynx</Tag>

[Rspeedy](https://lynxjs.org/rspeedy/) é uma ferramenta de build para aplicações Lynx.

### Re.Pack

<Tag color="geekblue">Ferramenta de build</Tag>

[Re.Pack](https://github.com/callstack/repack) usa Rspack para construir aplicações React Native.
